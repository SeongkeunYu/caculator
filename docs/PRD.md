# PRD — Calculator Class

## 1. 개요

| 항목 | 내용 |
|------|------|
| 문서 버전 | v1.0 |
| 작성일 | 2026-05-07 |
| 대상 | Calculator 클래스 개발자 |

단일 Calculator 클래스를 구현한다. 기본 사칙연산과 상태 관리를 제공하며, 잘못된 입력에 대해 명확한 예외를 발생시킨다.

---

## 2. 기능 요구사항

### REQ-001 — 덧셈
- 두 숫자를 입력받아 합산 결과를 반환한다.
- 정수 및 실수 모두 지원한다.

### REQ-002 — 뺄셈
- 두 숫자를 입력받아 차감 결과를 반환한다.
- 정수 및 실수 모두 지원한다.

### REQ-003 — 곱셈
- 두 숫자를 입력받아 곱셈 결과를 반환한다.
- 정수 및 실수 모두 지원한다.

### REQ-004 — 나눗셈
- 두 숫자를 입력받아 나눗셈 결과를 반환한다.
- 제수(divisor)가 0인 경우 예외를 발생시킨다.
- 결과는 실수로 반환한다.

### REQ-005 — 연산 이력 관리
- 수행한 연산을 순서대로 내부에 기록한다.
- 이력 조회 시 전체 연산 목록을 반환한다.
- 이력 초기화 기능을 제공한다.

### REQ-006 — 초기화
- 계산기를 초기 상태로 되돌리는 기능을 제공한다.
- 초기화 시 연산 이력도 함께 삭제된다.

---

## 3. 비기능 요구사항

### REQ-NF-001 — 입력 유효성
- 숫자가 아닌 값이 입력되면 예외를 발생시킨다.

### REQ-NF-002 — 불변 이력
- 이력 조회로 반환된 목록을 수정해도 내부 이력에 영향을 주지 않는다.

### REQ-NF-003 — 단일 책임
- Calculator 클래스는 연산과 이력 관리만 담당한다. UI·입출력 처리는 포함하지 않는다.

---

## 4. 인터페이스 명세

```
class Calculator:
    def add(a: int | float, b: int | float) -> int | float
    def subtract(a: int | float, b: int | float) -> int | float
    def multiply(a: int | float, b: int | float) -> int | float
    def divide(a: int | float, b: int | float) -> float   # b == 0 이면 예외
    def get_history() -> list                              # 연산 이력 반환 (복사본)
    def clear_history() -> None                            # 이력 초기화
    def reset() -> None                                    # 전체 초기화
```

## 5. 예외 정의

| 예외 | 발생 조건 |
|------|-----------|
| `ZeroDivisionError` | 나눗셈 시 제수가 0 |
| `TypeError` | 숫자가 아닌 값 입력 |

---

## 6. 이력 항목 구조

각 연산 이력은 아래 필드를 포함한다.

| 필드 | 설명 | 예시 |
|------|------|------|
| `operation` | 연산 종류 | `"add"` |
| `operands` | 입력 값 | `[3, 5]` |
| `result` | 연산 결과 | `8` |
